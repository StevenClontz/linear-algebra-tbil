Activity 3.6.3.

Let \(S=\{t^2,t^2-1,1+t\}\text{.}\) To determine whether or not \(t^2-t-1\) is an element of \(\vspan(S)\text{,}\) we must decide whether or not the vector equation
\begin{equation*} x_1t^2+x_2(t^2-1)+x_3(1+t)=t^2-t-1 \end{equation*}
is consistent.
(a)
Determine a system of linear equations in the variables \(x_1,x_2,x_3\) that helps you make this determination.
(b)
Is your system of linear equations consistent? If yes, express \(t^2-t-1\) as a linear combination of the polynomials in \(S\text{.}\)
(c)
Based on your calculations, what else can you conclude about the set \(S\text{?}\)
  1. The set \(S\) spans all of \(\P^3\text{.}\)
  2. The set \(S\) is linearly independent.
  3. The set \(S\) is a basis for \(\P^3\text{.}\)
  4. We cannot conclude anything further about \(S\) without additional information.
